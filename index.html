Cross-device sync

Syncs your watering log via a private GitHub Gist so checkboxes match on your phone and laptop.

First-time setup (step-by-step)
  1. Go to GitHub → Fine-grained tokens
  2. Click Generate new token
  3. Name: irrigation-dashboard; Expiration: 1 year (or longer)
  4. Repository access: public repositories (read-only) is fine — we don't need repo access, just Gist access
  5. Under Account permissions → Gists: set to Read and write
  6. Click Generate token, copy it, and paste below
  7. Leave the Gist ID blank on the first device — a new private Gist will be created
  8. On your second device: open this page, click the sync button, paste the same token, and paste the Gist ID shown on your first device (find it in the URL of the Gist, or it will be shown below after sync succeeds on device 1)
